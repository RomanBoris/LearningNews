# IntelliJ IDEA / Android Studio — Горячие клавиши (Windows)

## Код — форматирование и импорты

| Действие | Горячая клавиша |
|---|---|
| Удалить неиспользуемые импорты | `Ctrl + Alt + O` |
| Выровнять (форматировать) код | `Ctrl + Alt + L` |
| Переименовать (рефакторинг) | `Shift + F6` |

---

## Навигация

| Действие | Горячая клавиша |
|---|---|
| Перейти к классу/файлу | `Ctrl + N` |
| Перейти к файлу | `Ctrl + Shift + N` |
| Перейти к символу (методу, переменной) | `Ctrl + Alt + Shift + N` |
| Открыть недавние файлы | `Ctrl + E` |
| Перейти к строке | `Ctrl + G` |
| Перейти к декларации | `Ctrl + B` / `Ctrl + Click` |
| Назад / Вперёд (история переходов) | `Ctrl + Alt + ←` / `Ctrl + Alt + →` |
| Найти файл везде (Search Everywhere) | `Shift + Shift` |

---

## Редактирование

| Действие | Горячая клавиша |
|---|---|
| Дублировать строку | `Ctrl + D` |
| Удалить строку | `Ctrl + Y` |
| Переместить строку вверх/вниз | `Shift + Alt + ↑` / `Shift + Alt + ↓` |
| Закомментировать строку | `Ctrl + /` |
| Закомментировать блок | `Ctrl + Shift + /` |
| Выделить следующее вхождение слова | `Alt + J` |
| Мультикурсор (выделить всё) | `Ctrl + Alt + Shift + J` |
| Развернуть/свернуть блок кода | `Ctrl + -` / `Ctrl + +` |

---

## Поиск и замена

| Действие | Горячая клавиша |
|---|---|
| Найти в файле | `Ctrl + F` |
| Заменить в файле | `Ctrl + R` |
| Найти во всём проекте | `Ctrl + Shift + F` |
| Заменить во всём проекте | `Ctrl + Shift + R` |

---

## Запуск и отладка

| Действие | Горячая клавиша |
|---|---|
| Запустить (последняя конфигурация) | `Shift + F10` |
| Запустить в debug режиме | `Shift + F9` |
| Остановить | `Ctrl + F2` |
| Поставить/убрать breakpoint | `Ctrl + F8` |

---

## Рефакторинг

| Действие | Горячая клавиша |
|---|---|
| Переименовать | `Shift + F6` |
| Извлечь переменную | `Ctrl + Alt + V` |
| Извлечь функцию | `Ctrl + Alt + M` |
| Показать все рефакторинги | `Ctrl + Alt + Shift + T` |

---

## Быстрые действия

| Действие | Горячая клавиша |
|---|---|
| Quick Fix (исправить ошибку) | `Alt + Enter` |
| Показать параметры функции | `Ctrl + P` |
| Показать документацию | `Ctrl + Q` |
| Сгенерировать код (конструктор, toString...) | `Alt + Insert` |
| Окружить блоком (try/catch, if...) | `Ctrl + Alt + T` |

## Задание 2.3 — Первый Composable экран

Создай файл `presentation/home/HomeScreen.kt`.

**Требования:**
- Функция `HomeScreen(viewModel: HomeViewModel)` — принимает ViewModel
- Собирает `state` через `collectAsState()`
- `when (state)` — обрабатывает все три состояния:
  - `Loading` — показывает `CircularProgressIndicator()`
  - `Success` — показывает `LazyColumn` со списком заголовков (просто `Text` для каждой новости)
  - `Error` — показывает `Text` с сообщением об ошибке

Не нужен дизайн. Нужна структура.
